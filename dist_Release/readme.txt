GreenMulti 2 도움말

주의: 이 프로그램은 인터넷으로 넓은마을에 연결하기 때문에 V3이나 알약 같은 바이러스 백신 프로그램들이 바이러스로 오진하거나 사용을 제한할 수 있습니다. 
바이러스 백신에 의해 사용이 제한된 경우, 검사 예외 폴더로 GreenMulti2 폴더를 지정하시면 사용에 불편이 없을 것입니다.

1. 개발자 연락처 :  넓은마을 아이디 : philjang

2. 단축키

단축키가 기억나지 않으면 팝업키를 눌러 보세요. 
상하 방향키로 기능을 선택할 수 있습니다. 팝업메뉴에는 단축키도 읽어 줍니다. 기억하는 데에 도움이 될 것입니다.

* 공통
F1: 도움말 보기
메뉴/게시물 진입 : 엔터키
되돌아가기 : ESC키, Alt+왼쪽방향키, 
편집창이 없는 때에는 BackSpace키로도 되돌아 갑니다.
초기화면으로 이동: 알트 + Home키
다운로드 폴더 열기: Control + 영문 O키
로그인/로그아웃: Control + 영문 L키
코드 바로가기: Control + 영문 G키
파일 전송 정보 대화상자 열기: Control + 영문 J키
프로그램 종료: Alt + F4

* 게시물/메일 목록
새 게시물/메일 작성: 영문 W키
이전 페이지로 이동 : PageUp키
다음 페이지로 이동: PageDown키
파일 다운로드 : 영문 D키
게시물 검색: Control + 영문 F키

* 게시물/메일 본문 읽기
새 게시물/메일 작성 : 영문 W키
내가 쓴 게시물 수정: 영문 E키
쪽지 답장하기: 영문 R키
메일/쪽지/내가 쓴 게시물 삭제: Delete키
다음 게시물 보기: Alt + PageDown키
이전 게시물 보기: Alt + PageUp키
내용 읽기: * 게시물 본문/게시물 댓글/메일  본문/쪽지 본문에서 스페이스키.

* 파일 전송정보 대화상자
전송정보 갱신하기: 스페이스키
전송 취소하기: Delete

* 전송 정보 설명
첫번째 숫자는 프로그램에서 시도한 전송의 순번입니다.
앞에 '-'가 붙어 있는 것은 업로드입니다. '-' 없이 숫자만 있는 것은 다운로드입니다.
두번째는 파일 이름이며, 세번째는 전송률입니다. 네번째는 전송속도이며 다섯번째는 파일의 전체 크기입니다.
전소이 완료된 파일은 전송 정보 목록에서 자동으로 사라집니다. 모든 파일 전송이 마쳤다면 파일 전송정보대화상자가 텅 비게 됩니다.

3. '파일' 풀다운 메뉴
알트키를 누르면 '파일' 풀다운 메뉴가 열립니다. 아래 방향키로 내려가면 메뉴들이 나옵니다.
바탕화면 바로가기 만들기, 다운로드 폴더 변경, 메뉴 관리는 단축키가 지정되어 있지 않기 때문에 풀다운 메뉴로만 접근할 수 있습니다.

4. 기타 기능들
동시 다운로드 제한: 초록멀티2는 파일 다운로드를 동시에 3개까지 할 수 있습니다. 파일 업로드에는 제한이 없습니다. 
초록등대 정회원의 경우 조금 더 여유 있게 다운로드를 할 수 있습니다.
메일/쪽지 수신 알림: 새로운 메일이나 쪽지가 도착하면 음성으로 알려 줍니다.
상태표시줄에 현재 위치가 나타납니다. 
센스리더 단축키는 컨트롤 쉬프트 S입니다.
날짜시간 형식을 월 일 시 분으로 듣기 편하게 자동 수정합니다.
KBUMenu.dat 파일을 편집하면 원하는 게시판, 동호회 등을 추가할 수 있습니다.
단 0. 초록등대, 1. 공지사항, 2천 전자우편, 3. 게시판, 12. 전자도서관은 변경 불가능합니다. 이 다섯 가지는 오직 개발자만이 변경할 수 있습니다.

* 오류 발생 시
초록멀티2를 종료할 때 오류가 발생했다는 영문 메시지가 나올 수 있습니다. 
초록멀티2가 설치된 폴더에 보면
GreenMulti.exe.log
라는 텍스트 파일에 오류 메시지가 담겨 있습니다. 
이 파일의 내용을 '제작자에게...' 메뉴로 보내 주시면 초록멀티2 성능 향상에 큰 도움이 될 것입니다.
미리 감사드립니다.
